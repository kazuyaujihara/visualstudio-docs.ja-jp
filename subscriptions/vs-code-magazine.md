# <a name="code-magazine-included-in-visual-studio-subscriptions"></a>Visual Studio サブスクリプションに含まれる CODE Magazine 特典

CODE Magazine は、すべてのプラットフォームの開発テクノロジを網羅する、独立系ソフトウェア開発者向けの一流雑誌です。  選ばれた Visual Studio サブスクリプションでは、この雑誌の印刷版またはデジタル版のいずれかの 1 年間の購読を受け取ることができます。

## <a name="activation-steps"></a>アクティブ化の手順
CODE Magazine の購読を設定するプロセスは簡単です。  以下の手順を実行します。

1. [https://my.visualstudio.com/benefits](https://my.visualstudio.com/benefits?wt.mc_id=o~msft~docs) にアクセスします
2. ご利用のポータルの [プロフェッショナルな開発] または [教育] セクションで [CODE Magazine] タイルを見つけ、 **[コードの取得]** をクリックします。
   > [!div class="mx-imgBorder"]
   > ![[CODE Magazine] タイル](_img/vs-code-magazine/vs-code-magazine-tile.png "[CODE Magazine] タイル")
3. コードが取得されたことを示すメッセージが表示されます。  **[アクティブ化]** をクリックして続行します。  (アクティブ化する準備ができていなくても、問題ありません。  いつでもタイルに戻ってアクティブ化することができます。  コードを取得すると、[コードを取得] リンクが [アクティブ化] になります)。
   > [!div class="mx-imgBorder"]
   > ![CODE Magazine コードが取得されました](_img/vs-code-magazine/vs-code-magazine-success.png "コードが正常に取得されました")
4. CODE Magazine のランディング ページにリダイレクトされます。 お住まいの国を選択し、雑誌を受け取る方法を選択します。 
> [!NOTE]
> Visual Studio Dev Essentials メンバーがアクセスできるのは、デジタル版の購読のみです。 
5. 選択内容に基づいて、購読の価格と、適用されたクーポンの金額を示すメモが表示されます。
> [!NOTE]
> 印刷版の購読の場合:
> - 米国以外の購読者には、発送料金が発生する場合があります。 
> - 印刷版の初回配信には数週間かかります。
   > [!div class="mx-imgBorder"]
   > ![CODE Magazine 購読の詳細](_img/vs-code-magazine/vs-code-magazine-details.png "購読の詳細と価格")
6. 次に、購読者情報と配信アドレスを指定する必要があります  (配信アドレスは、フルフィルメント、オファー選択、およびデジタル アクセス プロセスをサポートしているため、デジタル版の購読にも必要です)。
7. 米国以外にお住まいで、雑誌の印刷版の購読を選択した場合は、送料をカバーするために請求先住所と支払い情報を入力するように求められることがあります。 
8. CODE HTML Newletter を選択するには、チェック ボックスをクリックします。
   > [!div class="mx-imgBorder"]
   > ![CODE Magazine 購読者情報](_img/vs-code-magazine/vs-code-magazine-subscriber-info.png "購読者情報と配送先住所")
9. **[注文]** をクリックして購読を完了します。  
購読注文が完了すると、確認ページが表示されます。このページには、受け取った号を確認するために使用できるリンク (https://codemag.com/my/fulfillment ) が表示されます。 

## <a name="eligibility"></a>特典を受ける条件
| サブスクリプション レベル                                                 |     チャネル                                            | 特長                                                          | 更新可能かどうか    |
|--------------------------------------------------------------------|---------------------------------------------------------|------------------------------------------------------------------|---------------|
| Visual Studio Enterprise (Standard)   | VL、Azure、リテール、一部の NFR<sup>1</sup> | 使用可能       |  いいえ          |
| Visual Studio Professional (Standard<sup>2</sup>) | VL、Azure、リテール                                       | 使用可能                                                            |  いいえ          |
| Visual Studio Test Professional (標準)                         | VL、リテール                                              | 使用可能                                             |  いいえ          |
| MSDN Platforms (標準)                                          | VL、リテール                                              | 使用可能                                              |  いいえ          |
| Visual Studio Enterprise、Visual Studio Professional (月間クラウド) | Azure | 使用できません | N/A |

<sup>1</sup>  *内容:Not for Resale (NFR)、Visual Studio Industry Partner (VSIP)、FTE、BizSpark、Imagine Premium が含まれます。除外:Microsoft Partner Network (MPN)、Most Valuable Professional (MVP)、Regional Director (RD)、MCT Software & Services Developer、MCT Software & Services、Imagine。*

> [!NOTE]
> Microsoft では、クラウド サブスクリプションの Visual Studio Professional 年間サブスクリプションおよび Visual Studio Enterprise 年間サブスクリプションが提供されなくなりました。 サブスクリプションの更新、増減、キャンセルに関する既存のお客様のエクスペリエンスと機能については変更はありません。 新規のお客様は、[https://visualstudio.microsoft.com/vs/pricing/](https://visualstudio.microsoft.com/vs/pricing/) に移動し、Visual Studio のさまざまな購入オプションを調べることをお勧めします。

どのサブスクリプション使用しているかわからない場合は次の手順を実行してください。  [https://my.visualstudio.com/subscriptions](https://my.visualstudio.com/subscriptions?wt.mc_id=o~msft~docs) に接続し、お使いのメール アドレスに割り当てられているすべてのサブスクリプションを確認します。 すべてのサブスクリプションが表示されない場合は、1 つ以上のサブスクリプションが別のメール アドレスに割り当てられている可能性があります。  それらのサブスクリプションを表示するには、そのメール アドレスを使用してサインインする必要があります。

## <a name="frequently-asked-questions"></a>よく寄せられる質問
### <a name="q-if-the-subscription-is-free-why-am-i-being-asked-for-a-credit-card"></a>Q:サブスクリプションが無料の場合に、クレジット カードを要求されるのはなぜですか?  
A: 印刷物を国外に無料発送するには多額のコストがかかります。  CODE Magazine の印刷版を選択した米国以外に居住する購読者には、発送料が請求されます。 発送料を回避するには、雑誌のデジタル版を選択してください。 

### <a name="q-why-do-i-need-to-provide-a-delivery-address-for-a-digital-subscription"></a>Q:デジタル版の購読に配送先住所を入力する必要があるのはなぜですか?
A: CODE Magazine では、フルフィルメント、オファー選択、およびデジタル アクセス プロセスをサポートするため、購読が印刷版またはデジタル版のどちらかに関係なく、住所を収集しています。  CODE Magazine がデータを使用して行っていることの詳細については、[プライバシー ポリシー](https://www.codemag.com/Home/Privacy)を参照してください。
