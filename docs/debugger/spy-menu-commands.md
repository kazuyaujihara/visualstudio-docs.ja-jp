# <a name="spy-menu-commands"></a>[スパイ] メニューのコマンド
**スパイ**メニューには、Spy++ は Windows とビューを操作するコマンドが用意されています。

 **Windows**新しい Windows ビューが開きます。

 **プロセス**新しいプロセス ビューを開きます。

 **スレッド**新しいスレッド ビューを開きます。

 **ログ メッセージ**新しいメッセージ ビューを開き、[メッセージ オプション ダイアログ ボックス](../debugger/message-options-dialog-box.md)ビュー。

 **[検索] ウィンドウ**が表示されます、 **FindWindow**マウスを使用してウィンドウを選択するダイアログ ボックス。

 **終了**spy++、ユーティリティを閉じます。
