# <a name="c6504"></a>C6504
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

警告 C6504: 無効な注釈です: プロパティは、ポインター、pointer-to-member、または配列型の値でのみ使用できます  
  
 この警告は、互換性のないデータ型のプロパティの使用を示します。 プロパティによってサポートされるデータ型の詳細については、次を参照してください。[注釈プロパティ](http://msdn.microsoft.com/f77b4370-6bda-4294-bd2a-e7d0df182a3d)します。  
  
## <a name="example"></a>例  
 次のコード生成のため、この警告は`_Null_`プロパティは、参照データ型では使用できません。  
  
```cpp  
#include<sal.h>  
  
class Point  
{  
public:  
    //  members  
};  
  
void f(_Pre_ _Null_ Point& pt)  
{  
    // code ...  
}  
  
```  
  
 この警告を解決するには、次のコードを使用します。  
  
```cpp  
#include<sal.h>  
  
class Point  
{  
public:  
    //  members  
};  
  
void f(_Pre_ _Null_  Point* pt)  
{  
    // code ...  
}  
```  
  
 前に示した欠陥のあるコードにも警告が生成[C6516](../code-quality/c6516.md)無効な注釈プロパティの競合が発生したためです。
