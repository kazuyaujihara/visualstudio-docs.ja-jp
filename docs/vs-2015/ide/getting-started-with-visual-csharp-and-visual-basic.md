# <a name="getting-started-with-visual-c-and-visual-basic"></a>Visual C# と Visual Basic の概要
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Visual C# または Microsoft Visual Basic を初めて使用する方は、ここがスタート ラインです。 このチュートリアル入門のヘルプは、Visual C# と Visual Basic について説明する一連のレッスンで構成されています。 楽しみながら、高機能のソフトウェアを開発してください。

 Visual C# について詳しくは、「[Visual C++ リソース](https://msdn.microsoft.com/vstudio/hh386302.aspx)」をご覧ください。

## <a name="in-this-section"></a>このセクションの内容
 [チュートリアル: 簡単なアプリケーションの作成](../ide/walkthrough-create-a-simple-application-with-visual-csharp-or-visual-basic.md)Visual Studio IDE で基本的なウィンドウを識別し、設定を構成して開発エクスペリエンスを最適化してから、簡単なプログラムをビルドします。

 [チュートリアル 1: ピクチャビューアーを作成する](../ide/tutorial-1-create-a-picture-viewer.md)ファイルから画像を読み込んでウィンドウに表示するプログラムをビルドします。 ボタンやピクチャ ボックスなどのコントロールをフォームにドラッグする方法、それらのプロパティを設定する方法、およびコンテナーを使用してフォームのサイズを滑らかに変更する方法を習得できます。 コードの記述の基本事項についても学習します。

 [チュートリアル 2: 時間指定の計算クイズを作成する](../ide/tutorial-2-create-a-timed-math-quiz.md)時間のかかる計算クイズゲームを作成します。プレーヤーは、指定された時間内に4つのランダムな算術問題に答える必要があります。 `Random` クラスを使用して乱数を生成する方法、`Timer` コントロールを使用してイベントを発生させる方法、`if else` ステートメントを使用してプログラム フローを制御する方法、および基本的な算術演算を実行する方法を習得できます。

 [チュートリアル 3: 絵合わせゲームを作成](../ide/tutorial-3-create-a-matching-game.md)するプレーヤーが非表示アイコンのペアと一致する必要がある絵合わせゲームを作成します。 `List` オブジェクトを使用してオブジェクトを保持する方法、`foreach` ループを使用する方法、参照変数を使用してフォームの状態を追跡する方法、複数のオブジェクトで使用できるイベント ハンドラーを構築する方法、および開始時にタイマーが一度だけ発生するようにする方法を習得できます。
