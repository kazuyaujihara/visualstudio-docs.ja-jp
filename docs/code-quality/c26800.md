# <a name="c26800"></a>C26800
C26800 を警告します。使用して、移動先のオブジェクトから:\<ロック >。

移動された後に変数を使用すると、警告 C26800 がトリガーされます。 変数は、右辺値参照として関数に渡された後から移動したと見なされます。 使用して割り当て、破棄、および std::vector::clear などの関数をリセットするいくつかの状態などのいくつかの正当な例外があります。

## <a name="example"></a>例
 次のコード C26800 が生成されます。

```cpp

#include <utility>

struct X {
    X();
    X(const X&);
    X(X&&);
    X &operator=(X&);
    X &operator=(X&&);
    ~X();
};

template<typename T>
void use_cref(const T&);

void test() {
    X x1;
    X x2 = std::move(x1);
    use_cref(x1);                // @expected(26800)
}
```

## <a name="example"></a>例
 次のコードでは、C26800 は生成されません。

```cpp

#include <utility>

struct MoveOnly {
    MoveOnly();
    MoveOnly(MoveOnly&) = delete;
    MoveOnly(MoveOnly&&);
    MoveOnly &operator=(MoveOnly&) = delete;
    MoveOnly &operator=(MoveOnly&&);
    ~MoveOnly();
};

template<typename T>
void use(T);

void test() {
    MoveOnly x;
    use(std::move(x)); // no 26800
}
```
