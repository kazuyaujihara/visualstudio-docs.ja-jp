# <a name="c6551"></a>C6551
C6551 を警告します。無効なサイズ指定です: 式が解析可能ではありません

> [!NOTE]
>  この警告は、ソース コード注釈言語 (SAL) の非推奨のバージョンを使用しているコードでのみ発生します。 最新バージョンの SAL を使用するコードを移植することをお勧めします。 詳細については、[C と C++ コードの欠陥を削減する SAL 注釈を使って](../code-quality/using-sal-annotations-to-reduce-c-cpp-code-defects.md)を参照してください。
