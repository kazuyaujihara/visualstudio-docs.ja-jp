# <a name="c28243"></a>C28243
C28243 を警告します。パラメーターの関数の注釈には、注釈が付けられた実際の型に許可された数よりも多くの逆参照が必要です

 数`__deref`演算子が注釈には、パラメーターの型によって定義されているポインターのレベル数よりも多い。 注釈または参照されているパラメーターのポインターのレベル数、いずれかを変更することでこの警告を解決します。
