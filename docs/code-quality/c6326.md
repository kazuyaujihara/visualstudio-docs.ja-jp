# <a name="c6326"></a>C6326
警告 C6326: ある定数と別の定数を比較している可能性があります

 この警告は、別の定数 (冗長コード) と定数を比較する可能性があることを示します。 目的がコードで適切にキャプチャされていることを確認する必要があります。 場合によっては、テスト条件を単純化して同じ結果を得ることができます。

## <a name="example"></a>例
 次のコードでは、2つの定数が比較されるため、この警告が生成されます。

```
#define LEVEL
const int STD_LEVEL = 5;

const int value =
#ifdef LEVEL
10;
#else
5;
#endif

void f()
{
  if( value > STD_LEVEL)
  {
    // code...
  }
  else
  {
    // code...
  }
}
```

 次のコードは、#ifdef ステートメントを使用して、どのコードを実行する必要があるかを判断することで、この警告を修正する方法の1つを示しています。

```
#define LEVEL
const int STD_LEVEL = 5;

const int value =
#ifdef LEVEL
10;
#else
5;
#endif

void f ()
{
#ifdef LEVEL
  {
    // code...
  }
#else
  {
    // code...
  }
#endif
}
```
