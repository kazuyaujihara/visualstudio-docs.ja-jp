# <a name="c28022"></a>C28022
警告 C28022:関数クラス\<classlist1 > この機能に一致しない関数クラス\<classlist2 > を定義するために使用された typedef。

 この警告は、注釈内でエラーがある場合に報告されます。 Typedef と、関数自体の両方が`_Function_class_`の注釈が、それらが一致しません。 両方を使用する場合、一致する必要があります。
