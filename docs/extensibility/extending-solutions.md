# <a name="extend-solutions"></a>ソリューションを拡張します。
ソリューションは、連携してアプリケーションを作成する 1 つまたは複数のプロジェクトのグループです。

## <a name="in-this-section"></a>このセクションの内容
 次のチュートリアルでは、ソリューションを拡張するいくつかの方法を説明します。

1. [ソリューション エクスプ ローラーのフィルターを拡張して](../extensibility/extending-the-solution-explorer-filter.md)ソリューション エクスプ ローラーの表示をフィルター処理する方法について説明します。

2. [管理ソリューションでプロジェクトの読み込み](../extensibility/managing-project-loading-in-a-solution.md)ユーザー ソリューションの読み込み時に、パフォーマンスを向上させるためにプロジェクトの読み込みを管理する方法について説明します。
