1. Visual Studio メニューで、 **[ファイル]**  >  **[新規作成]**  >  **[プロジェクト]** の順に選択します。
2. **[新しいプロジェクト]** ダイアログ ボックスの **[テンプレート]** セクションで、 **[Visual C#]**  >  **[Web]** の順に選択します。
3. **[ASP.NET Core Web アプリケーション]** を選択します。
4. 新しいアプリケーションの名前を入力したら (または既定の名前を使用)、 **[OK]** を選択します。
5. **[Web アプリケーション]** を選択します。
6. **[Docker サポートを有効にする]** チェック ボックスを選択します。
7. コンテナーの種類 (Windows または Linux) を選択し、 **[OK]** を選択します。
