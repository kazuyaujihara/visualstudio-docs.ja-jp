# <a name="options-text-editor-f-intellisense"></a>[オプション]、[テキスト エディター]、[F#]、[IntelliSense]

**[IntelliSense]** オプション ページを使用して、テキスト エディターでの F# 用のいくつかの設定を変更します。 このオプション ページにアクセスするには、**[ツール]** > **[オプション]** を選択し、さらに **[テキスト エディター]** > **[F#]** > **[IntelliSense]** の順に選択します。

## <a name="completion-lists"></a>入力候補一覧

- **文字が入力された後に入力候補一覧を表示する**

   このオプションを選択すると、入力を開始したときに IntelliSense によって入力候補一覧が自動的に表示されます。 このオプションを選択しない場合でも、IntelliSense の入力候補は、[IntelliSense] メニューから、または **Ctrl** + **Space** キーを押すことで使用できます。

- **文字が削除された後に入力候補一覧を表示する**

   このオプションを選択すると、コードの文字の削除を開始したときに、IntelliSense によって関連する入力候補一覧が自動的に表示されます。

- **開いていない名前空間内のシンボルを表示する**

   このオプションを選択すると、入力を開始したときに、開いていない名前空間の項目が含まれている場合は、IntelliSense によって入力候補一覧が自動的に表示されます。
