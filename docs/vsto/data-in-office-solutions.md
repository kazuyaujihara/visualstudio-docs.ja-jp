# <a name="data-in-office-solutions"></a>Office ソリューションのデータ

 Office アプリケーションを起動せずにアクセスできる埋め込みデータ アイランドにデータを格納する方法について説明します。

- [Office ソリューションのローカルデータベースファイルの使用の概要](../vsto/using-local-database-files-in-office-solutions-overview.md)
