# <a name="how-to-fill-listobject-controls-with-data"></a>方法: データを使用した ListObject コントロールの塗りつぶし
  データ バインディングを使用すると、ドキュメントにデータをすばやく追加できます。 リスト オブジェクトにデータをバインドした後、リスト オブジェクトを切断すると、データは表示されますが、データ ソースとのバインドは解除されます。
